# GiftLink - Full Stack Capstone Project

A full-stack web application for sharing and discovering gifts in the community.

## Project Links

- **Repository:** https://github.com/justinmarkdaniel/fullstack-capstone-project
- **Issues (Kanban):** https://github.com/justinmarkdaniel/fullstack-capstone-project/issues
- **Labels:** https://github.com/justinmarkdaniel/fullstack-capstone-project/issues/labels

## Tech Stack

- **Frontend:** React
- **Backend:** Express.js / Node.js
- **Database:** MongoDB
- **Containerization:** Docker
- **Orchestration:** Kubernetes
